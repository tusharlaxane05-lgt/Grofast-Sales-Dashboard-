##  Grofast Sales Dashboard (Power BI)

This project is an interactive **Sales Analytics Dashboard** created using Power BI. It provides insights into sales performance, item distribution, outlet types, and customer preferences.

###  Key Features:

* Total Sales Overview**: Displays total revenue of **$1.20M**
* Average Sales**: Shows average sales value per transaction
* Total Items Sold**: Around **9K items analyzed**
* Average Rating**: Customer rating insight (~3.92)

###  Visual Insights:

* Outlet Establishment Trend**: Year-wise growth of sales
* Outlet Size Distribution**: High, Medium, Small comparison
* Outlet Location Analysis**: Tier-wise sales performance
* Item Type Breakdown**: Sales across categories like Fruits, Snacks, Dairy, etc.
* Fat Content Analysis**: Comparison between Low Fat and Regular items

### Detailed Analysis:

* Sales contribution by:

  * Grocery Stores
  * Supermarket Type 1, 2, and 3
* Item-wise count and performance
* Category-wise ranking (Top: Fruits & Vegetables, Snacks)

### Purpose:

This dashboard helps businesses:

* Track sales performance
* Understand customer buying patterns
* Identify high-performing products and outlets
* Make data-driven decisions

* Power BI
